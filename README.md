<h1 align="center">
  Currency Mock Service
</h1>

A service that generates mock currency data at regular intervals!

## Usage
### Install
```bash
   git clone mehmetalidemirtas/socket.io-currency-mock-service
   cd socket.io-currency-mock-service
   ```
### Setup
```bash
   npm install
   ```
### Start
```bash
   node index.js
   ```
